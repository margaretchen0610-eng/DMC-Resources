# 图片资源说明

本目录包含网站所需的所有图片资源。

## 📁 目录结构

```
images/
├── team_photo.jpg          # 欢迎页面背景图（需自行添加）
├── balloon_images/         # 首页浮动气球照片文件夹
│   ├── balloon1.jpg
│   ├── balloon2.jpg
│   ├── ...
│   └── README.md          # 气球照片详细说明
└── README.md              # 本文件
```

## 🖼️ team_photo.jpg（欢迎页背景图）

### 用途
用作欢迎页面（`index.html`）的全屏背景图。

### 要求
- **文件名**：`team_photo.jpg`
- **格式**：JPG/JPEG
- **建议尺寸**：1920x1080 或更高分辨率
- **内容**：团队合照、活动照片或任何合适的背景图

### 注意
如果暂时没有团队照片，可以使用任何合适的背景图片作为占位。如果图片不存在，页面仍可正常显示（会显示遮罩层）。

## 🎈 balloon_images/（浮动气球照片）

### 用途
用于首页的浮动气球动画效果，展示团队成员照片或活动照片。

### 详细说明
请查看 `balloon_images/README.md` 了解：
- 照片命名规则
- 如何添加新照片
- 配置方法
- 照片建议

### 快速开始
1. 将照片命名为 `balloon1.jpg`, `balloon2.jpg`, `balloon3.jpg` 等
2. 放入 `balloon_images/` 文件夹
3. 在 `scripts/floating-balloons.js` 中配置文件名
4. 刷新浏览器即可看到效果

## 💡 提示

- 所有图片建议使用 `.jpg` 或 `.webp` 格式以减小文件大小
- 建议对图片进行压缩优化，提升加载速度
- 气球照片会被裁剪为圆形显示，建议使用正方形照片
